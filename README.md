## Dependencies

- cmake
- glfw-x11
- glew
- glm
- [assimp](https://github.com/assimp/assimp)


## Installation
```
git clone https://github.com/jervw/gl-demo.git
cd gl-demo
mkdir build && cd build
cmake ..
make
./opengl
```
